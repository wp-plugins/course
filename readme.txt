A plugin that will create a custom post type displaying courses offered by an institution.
== Description ==
We are pleased to present the WordPress plugin for creating a custom post type for the courses offered by some institution for the students. This Plugin provide the facility to display features like course type and its target to specify as custom fields as well as with featured image support. We hope that this will be a great addition in WP plugin directory for the users interested in such kind of development.
== Installation ==

Upload the Course plugin to your blog, Activate it.

1, 2: You're done!
